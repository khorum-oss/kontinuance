# C4 — Context & Container Views

Kontinuance's architecture follows the overview's three-plane split — Orchestrator,
Agent/Runner, Event Source — kept as separate concerns even though **v0 runs them in
a single process**. External systems and planes that arrive in later phases are
marked *(future)*.

## System Context

```mermaid
C4Context
    title Kontinuance - System Context
    Person(dev, "Developer", "Authors and runs CI/CD pipelines")
    System(kont, "Kontinuance", "Kotlin/Spring Boot CI/CD platform that runs user-defined pipelines")
    System_Ext(scm, "Source Control", "GitHub - triggers via webhooks (future)")
    System_Ext(cri, "Container Runtime", "Docker - step isolation (future)")
    System_Ext(db, "Database", "Postgres - run persistence (future)")
    Rel(dev, kont, "Defines pipelines, runs them, reads logs")
    Rel(scm, kont, "Push / PR events (future)")
    Rel(kont, cri, "Runs isolated steps (future)")
    Rel(kont, db, "Persists run state (future)")
```

## Container View (v0, single process)

```mermaid
C4Container
    title Kontinuance v0 - Container View (single process)
    Person(dev, "Developer", "Authors pipelines via YAML or Kotlin DSL")
    System_Boundary(k, "Kontinuance (v0, one process)") {
      Container(dsl, "Pipeline DSL", "Kotlin + Konstellation", "Hybrid YAML/Kotlin pipeline definition to pipeline model")
      Container(orch, "Orchestrator", "Kotlin coroutines", "Schedules stages/steps, tracks status FSM, concurrency cap")
      Container(agent, "Agent / Runner", "ProcessBuilder", "Executes steps in isolated workdirs, masks secrets, streams logs")
      Container(evt, "Event Source", "manual trigger (v0)", "Submits pipeline runs; webhooks/polling are future")
    }
    System_Ext(scm, "Source Control", "GitHub (future webhooks)")
    System_Ext(cri, "Container Runtime", "Docker (future isolation)")
    Rel(dev, dsl, "writes pipeline")
    Rel(evt, orch, "submits run")
    Rel(dsl, orch, "provides pipeline model")
    Rel(orch, agent, "dispatches steps")
    Rel(agent, dev, "streams masked logs")
    Rel(scm, evt, "events (future)")
    Rel(agent, cri, "runs containers (future)")
```
